# DOCKER Insatllation automated shell script for Ubuntu OS's.
 

* Download the script.

* Run the script with sudo.  

    ` ` ` sudo ./docker.sh ` ` ` 

* change the file permission with 777 

    ` ` ` sudo chmod 777 docker.sh ` ` `
   
* Latest current version of docker will be installed.

 ## Prerequisite:
    * Ubuntu OS.
